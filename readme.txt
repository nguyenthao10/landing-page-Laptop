1. Thư mục landing page
- css: thư mục chứa thư viện css của landing page
- fonts: thư mục chứa icon font của landing page
- img: thư mục hình ảnh của landing page
- js: thư mục chứa thư viện javascript
- index.html: file mã nguồn HTML của landing page
- style.css: file css chính của landing page

2. Tùy chỉnh landing page
- Bạn mở file index.html để chỉnh lại nội dung, hình ảnh, bố cục, ... của landing page.
- Bạn mở file style.css để thêm, chỉnh lại style cho landing page.
- Bạn mở file /js/main.js để thêm, chỉnh lại javascript cho landing page.
- Thay logo của bạn ở thư mục /img/logo/logo.png (logo có kích thước 171 x 39 px, nền trong suốt)
- Thay favicon của bạn ở thư mục /img/favicon.png (favicon có kích thước 44 x 44 px, nền trong suốt)
